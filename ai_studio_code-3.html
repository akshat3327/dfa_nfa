DFA DETERMINISTIC

A Deterministic Finite Automaton guarantees exactly one valid path for every sequence of inputs. No ambiguity, and no jumping between states without consuming input.

  • One unique transition per state per symbol.
  • No empty (epsilon / ε) transitions allowed.
  • Always knows exactly what state it is in.
M = (Q, Σ, δ, q₀, F)

δ: Transition function (Q × Σ → Q)

NFA NON-DETERMINISTIC

A Nondeterministic Finite Automaton explores multiple paths simultaneously. It branches into parallel realities to "guess" the correct path to an accepting state.

  • Can have 0, 1, or multiple transitions for a symbol.
  • Allows epsilon (ε) transitions (moving without input).
  • Accepts if at least one branching path succeeds.
M = (Q, Σ, δ, q₀, F)

δ: Transition function (Q × (Σ ∪ {ε}) → P(Q))
